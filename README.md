## Supported Features

##### Primitives

*  Elipse
*  Rectangle
*  Triangle

##### Visualization based on

* Object color
* Border color
* Border Thickness

##### Editor features

* Selection
* Translation
* Grouping
* Ungrouping
* Group Manipulation
* Selection from hierarchy
* Auto update on property change

##### UI

* Properties Pane
* Hierarchy Pane

##### Export & Import

* JSON (fully supported)
